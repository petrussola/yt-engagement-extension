Privacy Policy

Last updated: May 3, 2026

Data Collection

YouTube Engage-o-meter does not collect, transmit, sell, or share user data.

Data Processed Locally

The extension reads information that is already visible on the current YouTube watch page, such as view count, like count, comment count, and video age. This information is processed locally in the browser to calculate an engagement signal and render the on-page banner.

Storage

The extension may store local debug preferences in the browser so the user can show or hide debugging details. These preferences stay on the user's device and are not transmitted by the extension.

Network Requests

The extension does not send analytics, browsing history, page content, engagement counts, or personally identifiable information to any server.

Permissions

The extension uses a content script match for https://www.youtube.com/watch* so it can run on YouTube video watch pages. This access is necessary for the extension's core functionality.

Third Parties

The extension does not share data with third parties and does not include third-party tracking.

Contact

Questions or privacy requests can be opened as issues in the GitHub repository.
